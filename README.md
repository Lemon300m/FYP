# Background Deepfake Detector
## This program is entirely local based and will require your own dataset to train on.
To use this program, simply download the entire zip folder
1. Unzip it any where you want
2. Run the BUILD.bat and have it run entirely
(Optional: Delete all source file)
3. Gather two datasets of Real and Fake images in their respecitve folder
4. Open the exe then go to settings
5. Train the model (timing can vary depends on the file counts)
6. Start the system
